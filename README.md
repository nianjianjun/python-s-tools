##📖 项目名称：Python小白进化手记

一个记录从零开始挣扎到写出人生第一行 print("Hello World") 的野生代码日记。

魔性实验：用 turtle 画歪脖子树、用 requests 爬表情包等迷惑行为

黑历史存档：早期写的“屎山代码”与重构对比

⚙️ 技术栈

Python3 + 一堆百度来的库 + 玄学debug + 深夜咖啡因
